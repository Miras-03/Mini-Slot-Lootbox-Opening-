Мини—слот (открытие лутбокса) - Тестовое задание Unity

Реализован механизм открытия мини-слота / лутбокса с использованием фреймворка AxGridTools.


Ключевые функции:

-Архитектура на основе FSM (логика, управляемая состоянием: холостой ход → вращение → остановка → результат)

-Бесконечная вертикальная прокрутка барабана с повторным использованием объектов

-Плавное ускорение и замедление с точным выравниванием по центру при остановке

-Управляемое событиями взаимодействие между пользовательским интерфейсом, FSM и визуальным слоем

-Кнопки пользовательского интерфейса интегрированы с помощью привязки к данным (блокируются, когда они неактивны)

-Эффект динамического размытия в зависимости от скорости вращения барабана

-Определение результата на основе центрированного символа


Визуальная полировка:

-Система particle в зависимости от результата

-Обратная связь по результатам

-Адаптивная настройка пользовательского интерфейса для различных разрешений экрана (конфигурация Canvas)

Технический стек:

Unity, C#, AxGridTools (FSM, Model, Path), Система пользовательского интерфейса, система частиц
